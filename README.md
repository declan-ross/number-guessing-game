# Number Guessing Game (Python)

A command-line number guessing game written in Python.

## Features
- Easy, Medium, and Hard difficulty modes
- Difficulty-based number ranges
- Score tracking
- Replay system
- Input validation
- Error handling with try/except

## How to Run

1. Download the file
2. Open a terminal
3. Run the program with: python number_guessing_game.py

## Example Gameplay

The program randomly selects a number and the player must guess it within a limited number of tries.
